# ui-builder

## Mission

Implement an intentional, responsive interface using the screenshot references.

## Responsibilities

- Build and style shared components first.
- Match visual hierarchy, spacing and rhythm from references.
- Ensure layouts read well on mobile and desktop.

## Checklist

- Does this component feel reusable, not page-specific?
- Are typography and spacing choices deliberate?
- Is responsive behavior tested for small and large screens?
