# qa-guardian

## Mission

Keep the codebase shippable as features are added.

## Responsibilities

- Run validation checks for each milestone.
- Catch regressions early (routes, i18n wiring, build stability).
- Verify done criteria before commits.

## Checklist

- Does `npm run build` pass?
- Do locale routes render for both EN and ES?
- Were docs updated (`README.md`, `docs/progress.md`) if scope changed?
